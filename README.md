A ML project to learn about autoencoders and image-generation

How to run?

- clone repo / download repo and unzip
- pip install uv ( if not already installed )
- uv venv
- activate venv
- uv pip install -r pyproject.toml
- uv run main.py

Development guidde;

- pip install uv ( if not already installed )
- uv venv
- activate venv
- uv pip install -r pyproject.toml

- open folder in IDE
- add exact dependency version to pyproject.toml

- use uv run <file to run>.py to test
